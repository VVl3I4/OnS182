# OnS182

A Real-World Website Fingerprinting Dataset for Tor Onion Services.

We plan to partially release OnS182 after the formal publication of the associated paper. The release will be limited to a sanitized subset that supports academic reproducibility of the reported experiments. Raw or sensitive information that may introduce privacy, ethical, or operational risks will not be included.

Before publication, the dataset remains under embargo. After publication, this README will be updated with the dataset link, file format description, usage instructions, license terms, and citation information.
